# Esercizi Base Python per Search

Questa serie serve per consolidare alcune nozioni di Python,
ma su concetti che serviranno subito in BFS/DFS/A*.

Focus:
- liste e tuple per rappresentare stati
- cicli e condizioni
- set per visited/reached
- dict per parent e costi
- deque (FIFO) e list come stack (LIFO)
- queue.PriorityQueue
